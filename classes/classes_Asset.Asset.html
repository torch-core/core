Class Asset

Class representing an asset. Supports TON, Jetton, and Extra Currency. Implements Marshallable, Comparable, and Cellable interfaces.

Example

const tonAsset = Asset.ton()
const jettonAsset = Asset.jetton("EQC6...validAddress")
const jettonAsset = Asset.jetton(Address.parse("EQC6...validAddress"))

    const jettonAsset = Asset.jetton(Address.parse("EQC6...validAddress"))
